Home Page

This web application is written in JavaScript. It is a Single page application (SPA) backed by a RESTful web server.
JSON is used as the communications protocol between the SPA and the web server.
Token-based authentication is used for server side authentication.


References:

  • AngularJS as client side framework
  • ExpressJS as the web application framework
  • NodeJS as the server platform
  • MongoDB as the NoSQL database
  • Mongode as a thin wrapper around the MongoDB Node.JS driver
  • Bootstrap for CSS style sheets
  • Eclipse is used as the development IDE
  • GitHub is used for the source repository
